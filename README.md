## 📱 Published App

### [AskFiles — AI File Manager](https://play.google.com/store/apps/details?id=com.askfiles.mobile)
An AI-powered Android file manager with native performance.

**Tech stack:**
- React Native / Expo SDK 54 (New Architecture)
- Native Kotlin modules (MediaStore queries, Glide image rendering)
- Django REST Framework backend (Render)
- SQLite with BM25 full-text search
- RevenueCat (£2.99 lifetime Pro tier)
- Google Drive, OneDrive and Dropbox backup
- Android home screen widget
- Universal PIN + biometric app lock
